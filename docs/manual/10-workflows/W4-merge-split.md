# W4 · 合并候选 → 确认 / 拒绝 / 回滚

> status: 🟡 骨架
> 触发场景：审核中发现疑似重复视频 / 或定期清理合并候选

## 0. 元信息
- 涉及页面：[P-merge](../20-pages/P-merge.md) / [P-moderation](../20-pages/P-moderation.md)
- 适用角色：moderator+
- 关联设计：reference.md §5.9

## 1. 入口（多入口设计）
- 主入口：侧栏 → 合并拆分
- 从审核台进入：右栏「类似」Tab → 「发起合并」按钮 → 带 candidate_a 深链
- 从视频库进入：行 inline action「发起合并」→ 深链

## 2. 端到端步骤
```
①  待审候选 segment → 看置信度 pill 排序
②  card 形态：左右视频卡对比 + 中间合并原因 + 底部影响预览
③  「确认合并」→ 二次确认（哪个为主体）→ 执行
④  影响预览：线路 / 源 / 收藏 / 可回滚
```
