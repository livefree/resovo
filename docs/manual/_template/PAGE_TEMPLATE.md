## 6. 状态与颜色

> pill 颜色矩阵：ok=可用 / warn=需关注 / danger=必须处理。

| 颜色 | 状态 | 含义 |
|---|---|---|
| 绿（ok）| `已通过` / `已上架` | 可正常使用 |
| 黄（warn）| `待审` / `部分失效` | 需要关注 |
| 红（danger）| `已拒` / `全失效` | 必须处理或不可用 |
| 灰（muted）| `未测` / `待数据` | 信息缺失（**不是 mock**）|
