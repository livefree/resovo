Motion Spec · 可运行 demo

每张卡片点击 播放 即可演示一次动效。参数与 packages/design-tokens/src/primitives/motion.ts 对齐；PC 与移动端差异化的过渡各自单独列出。

1. 页面切换

PC 与移动端使用不同的过渡语言。reduced motion 模式下所有过渡退化为 80ms fade。

2. 交互微动效

hover / focus / 长按 / 下拉菜单 / tab 切换，时长多用 fast (120ms) 或 base (200ms)。

3. 播放与沉浸

Takeover（进播放）、MiniPlayer pop-in、Ken Burns 背景。

4. 加载与占位

Skeleton 闪烁 + 网格 stagger 进场，避免"空白-填充"的生硬。
