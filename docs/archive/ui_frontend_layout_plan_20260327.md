### 〇、 前端视觉与设计语言 (Design Tokens)

- **主色调与背景**：深邃背景（非纯黑，类似于 `#09090b` 的碳灰色），配合微弱的毛玻璃纹理背景（Grain effect）。
- **亮色点缀 (Accent)**：使用现代感的电光蓝 (Electric Blue) 或赛博紫/流光金，仅在按键、当前焦点（FocusRing）或进度条出现，克制且高级。
- **材质与边缘**：大量使用毛玻璃 (Glassmorphism, `backdrop-blur-md`)，如顶部导航栏悬浮时。卡片边缘使用极细的 1px 半透明边框（`border-white/10`），辅以圆角 (`rounded-xl` / `rounded-2xl`) 和平滑的悬停微动效特效（Hover Scale 1.02 + 泛光投影）。

### 一、 全局容器 (Global Layout)

**1. 顶部导航 (Sticky Navbar)**
高度约 64px，固定在顶部（随滚动增加毛玻璃背景底色）：

- `[左侧]`：Logo ➡️ 首页 ➡️ 电影 ➡️ 电视剧 ➡️ 动漫 ➡️ 更多 (Hover 下拉菜单，收纳综艺、纪录片、短剧等，保持顶部整洁)。
- `[右侧]`：搜索框 (点击展开或跳转) ➡️ ☀️/🌙 主题切换 ➡️ 圆形 Avatar (用户登录/中心入口)。

**2. 全局页脚 (Footer)**
精简极简风格，放置在 `min-h-screen` 底部容器：

- 包含 `Help`、`Privacy Policy`、`DMCA`、`关于我们` 等文字灰白链接，不会抢占主体视觉。

### 二、 页面布局结构设计

#### 1. 首页 (Home Page)

- **第一屏 (Hero Section)**：占据极大的屏幕视野。播放时下最热门的推荐视频大图（加上暗角渐变遮罩保护文本）。包含高亮度的主标题、豆瓣评分/年份标签，和一个明显的 `[▶ 立即播放]` 按钮。
- **第二/三屏 (Category Swimlanes 瀑布流/通栏)**：
  - 排列规则：“最新电影”、“热播剧集” 等横向滚动网格。
  - **卡片设计（含快捷入口）**：Hover（光标悬停）鼠标移上去时，卡片微缩放，中心浮现一个半透明的 `[▶ 播放]` 图标（直达 `/watch`），而点击海报其他区域则进入详情页 `/detail`。兼顾极速播放与 SEO。

#### 2. 分类/搜索页 (Browse & Search Page)

- **顶部筛选区 (Filters Bar)**：紧贴在导航栏下方。采用类似 Chip (胶囊) 的设计，横排罗列 `类型 (剧情/喜剧/科幻)`，第二排 `地区 (大陆/欧美)`，第三排 `年份`。点击后条件即刻高亮过滤。
- **下方网格矩阵 (Grid View)**：类似 Netflix 的平铺排布卡片。

#### 3. SEO 视频详情页 (Meta Detail Page, 参考 IMDB/Douban)

- **结构 (两柱型布局)**：
  - **左侧**：高清晰度海报立绘。
  - **右侧主体区域**：超大标题大字重，下方罗列评分（可以组合 TMDB 与 Douban 数据）、首播年份、电影时长。接着是卡司阵容（导演、主演）。
  - **行动区 (Call To Action)**：海报下方或标题旁显眼位置，放第一集的巨大播放入口大按钮，如果有选集可以在这里提供一个 Mini 的横排选集面板。
  - **下方文字屏**：较长的剧情简介摘要（Synopsis）等被搜索引擎爬虫高权重抓取的纯文本。

#### 4. 播放沉浸页 (Watch Page, 参考 YouTube 结构)

**极其关键的需求：多源+剧集切换既要近又不能影响视觉。**

- **顶部标题区**：在播放器**正上方**，以简洁的 `H1 视频标题 - 第 X 集` 形式呈现，紧凑大方。
- **左侧沉浸主视区 (Player Zone：占据 70%~75% 宽度)**：
  - 横向占满的 16:9 无边框/细黑框高级播放器。
- **右侧辅助区 (Interactive Sidebar：占据 25%~30% 宽度)**：
  - **顶部块：【选集 / 源切换 面板】**。这里用精致的折叠手风琴面板 (Accordion) 或并连 Tab 设计。默认展示当前季的分集；右上角一个小下拉框用来切换源（比如 `当前：主线HD` v）。紧挨播放器操作极短。
  - **下侧块：【相关推荐 (Up Next) / 稍后播放】**的短卡片列表。
- **播放器下方区 (Below Player)**：
  - 这里做 Tab 切换：`[详情介绍]` | `[弹幕列表/弹幕装填区]` | `[系统/用户评论区]`。
