# @resovo/player-core

Resovo 播放器核心层（零业务逻辑）。

## 用法

```tsx
import { Player } from '@resovo/player-core'
import type { PlayerProps } from '@resovo/player-core'
```

## 导出

- `Player` — 播放器主组件（原 YTPlayer）
- `PlayerProps` — 播放器 Props 类型
- `SubtitleTrack` — 字幕轨道类型
- `QualityLevel` — 画质选项类型
- `Chapter` — 章节类型

## 包含关系

此包通过 `git mv` 从 `apps/web/src/components/player/core/` 迁移而来，保留完整 git 历史。
